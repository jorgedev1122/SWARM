resumo do projeto

Combate rápido, ondas infinitas e boss fight progressivo.

SWARM foi feito utilizando

  • HTML5 Canvas
  • JavaScript
  • CSS3
  • Python

O jogo é um projeto rápido para praticar habilidades de desenvolvimento web, com foco em mecânicas de jogo, design de interface e otimização de desempenho. O objetivo é criar uma experiência divertida e desafiadora, onde o jogador enfrenta ondas de inimigos cada vez mais difíceis, culminando em uma batalha épica contra um boss final.
